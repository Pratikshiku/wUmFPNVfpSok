### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：普通用户、员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心
4. 修改密码
5. 添加业主信息
6. 费用信息、待缴费
7. 楼房信息
8. 报修信息
9. 车位信息
10. 我的停车信息
11. 投诉信息
12. 公告信息

**员工功能：**
1. 登录
2. 注册
3. 个人中心
4. 楼房信息管理
5. 报修信息管理
6. 投诉编号信息管理
7. 公告信息管理
8. 部门信息管理

**管理员功能：**
1. 登录
2. 个人中心
3. 用户管理
4. 员工管理
5. 业主信息管理
6. 费用信息管理
7. 楼房信息管理
8. 报修信息管理
9. 车位信息管理
10. 停车信息管理
11. 投诉编号管理
12. 公告信息管理
